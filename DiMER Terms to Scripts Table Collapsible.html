From Terms to Scripts: DiMER Chunking (Collapsible)

Tap a level to see its definition, asthma/COPD example, DiMER support, and working-memory impact.

1. Terms & Concepts Many separate facts; high intrinsic load.
+
Definition

Individual nodes: discrete facts, labels, or isolated concepts from each discipline.

Example from Asthma/COPD

β₂ receptor, airway smooth muscle, particle size, type 2 inflammation, ICS, LABA, DPI.

How DiMER supports this level

Each course (502, 503, 512) introduces discipline-specific terminology anchored to its primary DiMER element: Disease terms in 502, Mechanism terms in 503, Exposure/delivery terms in 512. Terms are no longer floating; they have a home in the D→M→E→R chain.

Working memory perspective: Many separate chunks; high intrinsic load when juggled simultaneously. Novices struggle to hold more than ~4–7 items.

2. Causal Links Pairs of nodes compressed into mini‑chunks.
+
Definition

Simple cause–effect pairs connecting two nodes (A → B).

Example from Asthma/COPD

ICS ↓ type 2 cytokines; LABA → smooth muscle relaxation; DPI needs adequate inspiratory flow.

How DiMER supports this level

Students practice explicit D→M, M→E, and E→R links via the three hypotheses, creating repeated cause–effect pairs that become increasingly automatic.

Working memory perspective: Each link reduces two separate chunks to one mini‑chunk, slightly lowering load if links are well-learned.

3. Mechanistic Subgraphs Runnable networks that can be simulated.
+
Definition

Small, runnable causal networks linking multiple nodes across several steps; can be mentally simulated forward or backward.

Example from Asthma/COPD

Allergen → epithelial injury → type 2 cytokines → eosinophils → airway edema → obstruction; ICS acts on cytokine production, LABA acts on smooth muscle tone.

How DiMER supports this level

Each week's case requires running the full D→M→E→R chain for one disease, building reusable subgraphs (for example, “how ICS–LABA works in type 2 asthma from mechanism to delivery to response”).

Working memory perspective: A practiced subgraph becomes a single larger chunk in working memory (for example, “ICS–LABA mechanism in type 2 asthma”), dramatically reducing apparent element count.

4. Schema Abstract patterns with slots and roles.
+
Definition

Abstracted organizational pattern that specifies slots (for example, “irritant,” “target,” “barrier”) and typical relationships; generalizes across cases.

Example from Asthma/COPD

Reversible airflow obstruction schema: smooth muscle hyperresponsiveness + inflammation + mucus → variable symptoms; responsive to bronchodilators and anti-inflammatory agents.

How DiMER supports this level

DiMER itself is a high-level schema (“every therapy story is D→M→E→R”) that students apply repeatedly across diseases, progressively refining their understanding of how disease, mechanism, exposure, and response relate.

Working memory perspective: A schema retrieved from long-term memory occupies only one “slot” in working memory, even though it encodes many relationships; this frees capacity for reasoning and problem‑solving.

5. Scripts Illness, therapy, and MTP patterns in context.
+
Definition

Schema embedded in clinical context, with enabling conditions, typical trajectories, cues, actions, and outcomes; includes illness scripts, therapy scripts, and MTP scripts.

Example from Asthma/COPD

Eosinophilic asthma illness script: young onset, atopy, variable symptoms, reversible obstruction, eosinophilia → responds well to ICS–LABA; therapy script includes device choice, monitoring FeNO/eosinophils, step‑up/down rules.

How DiMER supports this level

Repeated, varied cases using the D→M→E→R scaffold accelerate formation of:

  • Illness scripts (D + parts of M),
  • Therapy scripts (D→M→E→R for standard regimens),
  • MTP scripts (breaks in the D→M→E→R chain).

Working memory perspective: A mature script is a very large chunk retrieved as a unit; experts activate a script, then selectively unpack details as needed, keeping intrinsic load low even in complex cases.

Tap through the levels in order during class to make the progression from “many small chunks” to “a few powerful scripts” concrete for students and colleagues.

Teaching tip

On phones or tablets, you can ask participants to open only one level at a time and summarize, in their own words, how DiMER reduces working‑memory load at that stage. Then have them contrast Level 1 and Level 5 to make the chunking story visible.
